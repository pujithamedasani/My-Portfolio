#  Pujitha Medasani - Personal Portfolio

A modern, minimal, and responsive portfolio website designed to showcase my work as a **Software Developer Intern** and **AI Intern**. This project features a dark-themed UI, smooth animations, and a comprehensive look at my technical expertise in Backend Development, Data Analysis, and Systems Programming.

##  Built With

- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styling using Flexbox, Grid, and CSS Variables for a dark-mode aesthetic.
- **JavaScript (Vanilla):** Interactive typing effects, smooth scrolling, and form handling.
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.

##  Key Features

- **Modern Dark UI:** A charcoal and neon-teal aesthetic for a professional tech-focused look.
- **Dynamic Typing Effect:** An engaging hero section that highlights core roles.
- **Project Showcase:** A clean grid layout featuring 5 major projects across Web Dev, AI, and C++.
- **Smooth Navigation:** Seamless transitions between sections via CSS scroll-behavior.
- **Contact Integration:** A functional contact section ready for backend integration (e.g., Formspree).
